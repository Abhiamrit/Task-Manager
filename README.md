# To-Do List Web App

A simple and responsive task manager that allows users to add, complete, and delete daily tasks.

## Live Demo
https://abhiamrit.github.io/To-Do-List/

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
